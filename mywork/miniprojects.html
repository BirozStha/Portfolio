Project 1: Calculator

A clean, responsive calculator built with HTML, CSS, and JavaScript that performs basic arithmetic operations (addition, subtraction, multiplication, division) with real-time display and error handling.



Project 2: To-Do List

A dynamic to-do list app built with HTML, CSS, and JavaScript. Features task creation/deletion, LocalStorage persistence, and a responsive design. Perfect for daily task tracking with intuitive controls.
It also has light and dark mode the first image is light and second image is dark mode.



Project 3: Password Generator

A modern password generator built using HTML, CSS, and JavaScript. Offers customizable options like length selection, character type inclusion (uppercase, lowercase, numbers, symbols), and real-time strength feedback.
The app features a responsive design and includes a convenient copy-to-clipboard button for quick access to generated passwords.
